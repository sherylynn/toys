## 🔧 API接口文档

### 1. 价值投资分析
```
GET /api/value-investment/:stockCode
```
获取完整的价值投资分析报告

### 2. 股票基本信息
```
GET /api/stock-info/:stockCode
```
获取股票基本信息

### 3. 财务指标
```
GET /api/financial-metrics/:stockCode
```
获取财务指标数据

### 4. 批量分析
```
POST /api/analyze-value
```
批量分析多只股票

## 🎯 核心特性

### 价值投资评分算法
- **P/E比率评分**（20分）：15-25倍为最佳
- **ROE评分**（20分）：15%以上为优秀
- **负债率评分**（15分）：低于30%为健康
- **流动比率评分**（10分）：大于1.5为健康
- **成长性评分**（20分）：营收和利润增长
- **股息率评分**（5分）：3%以上为优秀

### 投资建议标准
- **80分以上**：强烈建议买入
- **60-79分**：建议买入
- **40-59分**：建议持有
- **20-39分**：建议观望
- **20分以下**：建议回避
